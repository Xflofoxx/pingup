Commit Message Template

- Type: feat | fix | docs | style | refactor | test | chore
- Scope: area of the codebase (e.g., scanner, api, cli)
- Subject: short, imperative description
- Body: detailed description, including why and what changed
- Footer: closes #ISSUE or related links

Example:
feat(scanner): add CIDR support

Add CIDR range input handling to the scanner and update tests.

- Closes #123
