# SPEC Glossary

- SPEC: Specification for how AI agents should operate and be governable.
- Alignment score: A numeric indicator (0-1) of how closely an AI output adheres to SPEC.
- Gate: A stage in CI/CD that enforces compliance with SPEC (lint, tests, validator).
- Drift: Deviation of an agent's output from the SPEC baseline.
- Artifact: Any file produced by an agent (code, docs, configs).
- Validator: Tool that checks artifacts against the SPEC.
